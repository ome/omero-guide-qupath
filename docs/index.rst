QuPath
======

QuPath is a free open-source cross-platform software application designed for bioimage analysis - and specifically to meet the needs of whole slide image analysis and digital pathology. We demonstrate how to integrate QuPath and OMERO using the User Interface. For more details, go to `QuPath Wiki page <https://github.com/qupath/qupath/wiki/>`_.
